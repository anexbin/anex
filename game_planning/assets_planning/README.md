# Assets Planning

**Purpose:** Track all visual and audio assets for the game.

## Why this folder exists
- Helps organize images, sounds, and fonts per arena, card, or UI.
- Ensures all assets are planned before coding.
- Makes future development and asset loading easier.

## Suggested files
- graphics.md → Arena backgrounds, card art, icons
- sounds.md → Card actions, UI notifications, background music
- fonts.md → UI fonts and styles

## Notes
- Include priority for asset creation.
- Track asset sources and references.
