# Leagues

**Purpose:** Plan progression through leagues and rewards for players.

## Why this folder exists
- Leagues structure player advancement beyond arenas.
- Allows planning XP milestones, unlocks, and difficulty balancing.
- Helps prevent exploit or progression issues before coding.

## Suggested files
- league1.md → Beginner rewards and AI difficulty.
- league2.md → Intermediate challenges.
- anex_league.md → Advanced league with complex mechanics.

## Notes
- Track skills or mechanics needed to progress.
