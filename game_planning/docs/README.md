# Docs

**Purpose:** Store game design notes, sketches, rules, and bug ideas.

## Why this folder exists
- Central knowledge base for the game.
- Captures planning, design decisions, and sketches.
- Useful for future reference or sharing with collaborators.

## Suggested files
- game_design_notes.md → Core rules, concepts, feature ideas
- sketches.md → Pen-and-paper or digital mockups
- bug_ideas.md → Track known or potential issues

## Notes
- Keep this folder lightweight but detailed enough to reference quickly.
