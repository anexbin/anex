# Backend Logic

**Purpose:** Plan XP, tokens, leaderboard, and ranking systems.

## Why this folder exists
- Separates gameplay from business logic.
- Helps prevent logic exploits (XP farming, leaderboard cheating).
- Prepares a blueprint for coding secure backend systems.

## Suggested files
- xp_and_levels.md → Leveling formulas and XP rewards
- tokens.md → In-game currency/access planning
- leaderboard.md → Ranking system design
- ranking_system.md → Win/loss and match calculations
