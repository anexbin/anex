# UI Design

**Purpose:** Plan all user interface screens, widgets, and player interactions.

## Why this folder exists
- Allows visual and interaction planning before coding.
- Reduces UI trial-and-error.
- Centralizes UX decisions for menus, profile screens, and widgets.

## Suggested files
- screens.md → Menu, Profile, Settings, Arena layouts
- widgets.md → XP bars, badges, buttons, notifications

## Notes
- Include mockups or sketches if helpful.
